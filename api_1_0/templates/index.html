About

Use Actionist to find and complete social justice actions on the go:

  • Add an event to your calendar
  • Call an Elected Official
  • Email an Elected Official

Actionist is an iOS app and REST API created by Kate Shaffer as an Ada Developers Academy Capstone Project. The iOS app is built in JavaScript with React Native. The API is built in Python using Flask, SQLAlchemy, Alembic, and Flask-HTTPAuth. It's hosted on AWS Elastic Beanstalk with a Postgres database.
